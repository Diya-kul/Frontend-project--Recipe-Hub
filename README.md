# Frontend-project--Recipe-Hub
Recipe Booklet Web App is a simple and interactive web-based cookbook that showcases easy recipes from around the world. The project is built using HTML, CSS, and JavaScript, focusing on clean UI design and user-friendly navigation. Users can explore different recipes, view ingredients, and follow step-by-step procedures in a book-style layout.

A simple and elegant web-based cookbook that displays easy recipes from around the world 🌍.

---

## 🚀 Features

- 📖 Book-style UI (clean and minimal)
- 🌍 Recipes from different countries
- 🥗 Ingredients & step-by-step procedure
- 💻 Fully responsive design
- 🎨 Clean and reusable CSS

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
